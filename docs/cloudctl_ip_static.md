## cloudctl ip static

make an ephemeral ip static such that it won't be deleted if not used anymore

```
cloudctl ip static <ip> [flags]
```

### Options

```
      --description string   set description of the ip address [required]
  -h, --help                 help for static
      --name string          set name of the ip address [required]
```
