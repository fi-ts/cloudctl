## cloudctl cluster issues

lists cluster issues, shows required actions explicitly when id argument is given

```
cloudctl cluster issues [<clusterid>] [flags]
```

### Options

```
  -h, --help               help for issues
      --id string          show clusters of given id
      --name string        show clusters of given name
      --partition string   show clusters in partition
      --project string     show clusters of given project
      --tenant string      show clusters of given tenant
```
